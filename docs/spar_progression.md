## 机制要点
- 野兽段（岩蛭/巨型犬）：cps 5~15 → 出手频率≈人类2倍；不说话不浪费心跳
- 避雷：蝴蝶/蜜蜂/金丝雀（不出手，实测12-19/h）、店小二（dp 9M摆设）、
  衙役（法庭执法）、黑白无常（死亡区）、一切持武器NPC（真伤）
- 留活口策略：对手濒死即换驻点（打死=等800s reset；留活=对手快速回血可持续）
- 多副本=闪避/被重击通道线性倍增，但 PARALLEL_SPAR 受 eff_kee 约束
